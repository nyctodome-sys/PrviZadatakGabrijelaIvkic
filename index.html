Popis elemenata korištenih u projektu

U ovoj vježbi koristila sam razne HTML i CSS elemente. HTML struktura omogućuje semantičko označavanje, dok CSS definira izgled i raspored elemenata na stranici. Sada za treću vježbu dodali smo i elemente kao HTML oznake: form, fieldset, legend, label, input, select, datalist, option, button HTML atribute: id, name, for, type, value, placeholder i CSS svojstva za oblikovanje i obojenje predstavljena kroz prethodne vježbe. Također sam dodala i neke druge elemente stranici kao dodatak ;)

Nenumerirani popis

  • Semantičke oznake (header, footer, section)
  • Tipografija pomoću Google Fonts
  • Prilagođene boje i naglašeni elementi

Numerirani popis

  2. Dodavanje fonta Roboto Condensed
  3. Stiliziranje pomoću CSS svojstava
